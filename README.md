## Getting Started

Simply Open the App or Feature you wish to learn by selecting a folder of your choosing.

1. Clone the repo

```sh
git clone https://github.com/FalseG0d/PythonPaint.git
```

2. Install Dependencies

```sh
pip install -r requirements.txt
```

3. Run index.py file.

```sh
python index.py
```
